Easy Install

Windows

  1. Download SARCAT package
  2. Place your File in a directory of your choice and unzip
  3. Inside the directory, click mongodb-win32-i386-3.0.4-signed.msi to install MongoDB into sarcat/bin/mongodb
  4. Inside the directory, click node-v0.10.36-x86.msi to install Node.js into sarcat/bin/node
  5. On the bottom left corner of the screen, click Run and type cmd into the text input
  6. Navigate to the SARCAT directory (ex: dir C:\Program Files\sarcat)
  7. Run SARCAT by typing bin/node/bin/node index.js

Mac/Linux

  1. Download SARCAT package
  2. Place your File in a directory of your choice and unzip
  3. Open you Terminal by typing Terminal into the search bar
  4. Navigate to the SARCAT directory (ex: cd /Users/KyleK/github/sarcat)
  5. Run SARCAT by typing ./run.sh

Install Node & Mongo From Source

1. Install Node.js v0.10.6

SARCAT is built with Meteor - which makes it fast and reliable. Meteor has been tested up to node version 0.10.36. If you are building from source, NVM is a great way to use bash scripting to manage multiple active node.js versions.

Just click the link and follow install instructions for your platform.

  • Node for Mac
  • Node for Linux

Mongo is the powerful database behind SARCAT. Follow the isntructions to get up and running with mongo! SARCAT has been tested with the latest version: v3.0.4.

    Troubleshooting

    SARCAT is intended to be used as a web server. If installing on a local network, you may need to contact your administrator to be granted access rights to install. While the steps and dependencies are minimal for this software, proper knowledge of your network is important to ensure quality.

    • Ensure proper priveleges are set on your machine and install paths
    • If you have problems with the binary installs, try installing from source
    • Feel free to contact us with any question or concerns you may have. This is an open source project and we encourage collaboration in order to create the best product possible!
